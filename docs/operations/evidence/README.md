# Operations Evidence

Canonical location for saved verification artifacts.

- Milestone 5 verification snapshots:
	- `docs/operations/evidence/milestone-5/<YYYY-MM-DD>/`
- Canonical archive command:
	- `HOSTED_BASE_URL=https://congress-portfolio-web.onrender.com DATABASE_URL='<render-postgres-connection-string>' ./scripts/ops/run-and-archive-hosted-m5.sh`

Operator checklist artifact requirements:
- Latest archived hosted verification output:
	- `m5-verification-<timestamp>.txt`
	- `m5-verification-<timestamp>.json`
	- `m5-verification-<timestamp>.status.json`
- Latest SQL snapshots:
	- ingestion summary query output
	- pricing refresh summary query output
- Latest closure or sign-off note:
	- `milestone-5-closure-note-<YYYYMMDD>.md` in the date folder
